`gonduit` is the Go library for working with Phabricator Conduit API [1].

*WARNING*: This library use new (or unstable) API of Conduit.

## Features

* Project
  * Create
  * Search

For examples, see the `*_test` files.

[1] https://secure.phabricator.com/conduit/
